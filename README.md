# Object-Oriented Programming Language and Lab (CSE 121 & 122)

Welcome to the repository for **CSE 121 & 122: Object-Oriented Programming Language and Lab**! This repository serves as a structured collection of my coursework, practical assignments, and theoretical implementations written entirely in **C++**. 

As a B.Sc. Engineering student at **BUBT**, I created this repository to document my academic progression through core OOP concepts, from foundational lecture notes to practical lab exercises and group presentations (such as our teamwork with PUNGDOS). 

## 📂 Repository Structure

The repository is organized into distinct directories to easily navigate between theoretical concepts and hands-on coding tasks:

- **`/THEORY CLASS`**: Contains code snippets, examples, and concepts covered during lecture sessions. This focuses on understanding the core pillars of OOP: Encapsulation, Inheritance, Polymorphism, and Abstraction.
- **`/LAB`**: Includes practical lab assignments, hands-on coding exercises, and problem-solving tasks implemented in C++. 
- **`/Circle Class`**: A dedicated module showcasing a specific implementation of a `Circle` class, demonstrating object creation, constructors, methods, and access modifiers.
