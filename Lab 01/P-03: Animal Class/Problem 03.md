# Animal Class–Object

## Problem Statement
Define a class named **Animal** with the following attributes:
- `name`
- `color`
- `sound`

Implement the following methods:
- **printinfo()** – displays the animal’s color and sound  
- **showname()** – displays the animal’s name  

Create an object of the `Animal` class and use these methods to display the animal’s details.

## Input
No input required.

## Output
Display the name, color, and sound of the animal.
