# My Car Class–Object Problem

## Problem Statement
Define a class named **Car** with the following attributes:
- `brand`
- `model`
- `year`

Implement the following method:
- **displayinfo()** – displays the car’s brand, model, and year.

Create an object of the `Car` class and use the method to display the car’s details.

## Input
No input required.

## Output
Display the brand, model, and year of the car.
