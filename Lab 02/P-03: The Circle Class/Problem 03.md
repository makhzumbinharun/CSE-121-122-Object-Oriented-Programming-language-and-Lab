# Problem 03: The Circle Class

## Description
Design a class named `Circle` that represents a geometric circle. The class must encapsulate the circle's radius and provide functionality to compute its area and circumference.

## Class Specifications
1.  **Attributes** (Private):
    * `radius`: A floating-point variable (double) to store the radius.
2.  **Methods** (Public):
    * `setRadius(double r)`: Assigns a value to the radius.
    * `getArea()`: Calculates and returns the area.
    * `getCircumference()`: Calculates and returns the circumference.

## Task
In your `main` function:
1.  Create an object of the `Circle` class.
2.  Set the radius using user input or a fixed value.
3.  Display the Area and Circumference.

## Formulas
Assume $\pi \approx 3.1416$
$$\text{Area} = \pi \times r^2$$
$$\text{Circumference} = 2 \times \pi \times r$$
