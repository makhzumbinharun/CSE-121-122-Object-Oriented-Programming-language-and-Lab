# Problem 05: The Book Class

## Description
Design a class named `Book` that represents a book in a library system. The class must store specific metadata about the publication and provide a way to display this information to the user.

## Class Specifications
1.  **Attributes** (Private):
    * `title`: A string representing the book's title.
    * `author`: A string representing the author's name.
    * `publicationYear`: An integer representing the year of publication.
2.  **Methods** (Public):
    * **Constructor** or **Setter**: To assign values to the attributes.
    * **`displayDetails()`**: To print the book's title, author, and year in a formatted manner.

## Input Format
1.  A string for the **Title** (may contain spaces).
2.  A string for the **Author** (may contain spaces).
3.  An integer for the **Year**.

## Output Format
Print the details of the book as shown in the sample.
